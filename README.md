# koodilehto-crm-frontend

Frontend of Koodilehto CRM.

## Development

1. `npm install`
2. `npm run hot-dev-server` and wait for the first compilation to finish
3. `npm run start-dev` at another terminal
4. Surf to `http://localhost:8080/`
